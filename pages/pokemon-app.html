Pokémon App

Esta es una aplicación web interactiva que permite explorar una lista de Pokémon con funcionalidades avanzadas.

Características principales:

  • Lista interactiva de Pokémon
  • Filtrado por tipo, nombre, etc.
  • Paginación para navegar fácilmente
  • Autenticación de usuarios
  • Desplegado en Firebase

Tecnologías utilizadas:

React, TypeScript, HTML, CSS, Firebase, Vite
