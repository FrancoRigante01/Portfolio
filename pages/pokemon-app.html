Pokémon App

Esta es una aplicación web interactiva que permite explorar una lista de Pokémon con funcionalidades avanzadas desarrollada con React y TypeScript.

Características principales

🎮 Interfaz Interactiva

Lista dinámica de Pokémon con navegación fluida y diseño responsive

🔍 Filtrado Avanzado

Busca por nombre, tipo, generación y otras características específicas

📄 Paginación

Navegación eficiente con sistema de páginas para mejor rendimiento

🔐 Autenticación

Sistema de usuarios con Firebase Authentication

🚀 Despliegue

Aplicación desplegada en Firebase Hosting con CI/CD

📱 Responsive

Diseño adaptativo que funciona en todos los dispositivos

Tecnologías utilizadas

React TypeScript HTML5 CSS3 Firebase Vite Git
